* [NOTE] How to use "QMK CLI"
PIP packages was strict about install on Ubuntu 22.04 a.k.a "Jammy Jellyfish" and later.
I'm suggest, "Using =python3 -m venv= and =pip= in that virtual environment".

Here directory include =qmktool= whose =requirements.txt= and =bin/activate.*= . for example;

#+begin_src
source ./qmktool/bin/activate.sh
pip install -r ./qmktool/requirements.txt
qmk setup
qmk flash ./dvorak_and_jis_mixed_rev12.json -- # After, standby frashing...

deactivate
#+end_src
